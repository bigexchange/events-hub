⚙ Sync Settings

Paste your deployed Web App URL here to sync events to Google Sheets and enable Slack + Calendar automations.
  1. Open script.google.com → New Project
  2. Paste the contents of events-backend.gs
  3. Edit the config vars at the top (Slack webhook, email, Calendar ID)
  4. Deploy → New deployment → Web App → Execute as: Me → Access: Anyone
  5. Copy the URL and paste it above
  6. For daily overdue alerts: Triggers → Add Trigger → checkOverdueTasks → Day timer
